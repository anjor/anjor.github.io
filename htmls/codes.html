Gandalf

Gandalf is a new Fourier-Hermite spectral code that solve the Kinetic Reduced MHD equations that describe the evolution of Alfvenic and slow mode turbulence in weakly collisional, magnetized plasmas at scales larger than the ion Larmor radius.

Gandalf is a highly optimized parallel code written in CUDA, a GPU computing platform and programming model by NVIDIA.



Viriato

Viriato is a Fourier-Hermite spectral code for magnetized plasma turbulence. Viriato solves the KREHM model, a 4D reduced gyrokinetic model valid for low plasma beta, as well as the KRMHD model for kinetic long wavelength slow mode equations.

Visit the Viriato homepage to find out more.
